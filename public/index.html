📚 Sistema Multi-Lezione

Questo corso è organizzato in lezioni multiple con contenuti dinamici:

  • 🏗️ Lezione 1: Fondamenti di Architettura Web (25 sezioni)
  • 🚀 Lezione 2: Architetture Avanzate (4 sezioni)

ℹ️ Come Funziona

Questa è una Single Page Application (SPA) con caricamento dinamico:

  • Selettore Lezioni: Dropdown in alto per cambiare lezione
  • Navigazione: Menu a sinistra per le sezioni della lezione corrente
  • Accessibilità: Frecce ← → per precedente/successivo
  • Progress: Barra aggiornata in base alla sezione corrente
  • Tema: Premi T oppure usa il toggle in alto
