💻 Installazione Node.js

Guida step-by-step per installare Node.js e preparare l'ambiente di sviluppo.

📥 Metodi di Installazione

🌐 Download Ufficiale

Più semplice per principianti

# Visita nodejs.org
# Scarica LTS version
# Installa seguendo wizard
# Verifica installazione:
node --version
npm --version

📦 Package Manager

Ubuntu/Debian:

curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

macOS (Homebrew):

brew install node

🔄 NVM (Consigliato)

Node Version Manager - gestione multiple versioni

# Installa NVM
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# Ricarica shell
source ~/.bashrc

# Installa Node.js LTS
nvm install --lts
nvm use --lts

🐳 Docker

Ambiente isolato e riproducibile

# Pull immagine ufficiale
docker pull node:18-alpine

# Run container interattivo
docker run -it --rm -v $(pwd):/app node:18-alpine sh

✅ Verifica Installazione

Check versions: node --version && npm --version

Test Hello World: echo "console.log('Hello Node.js!')" > hello.js && node hello.js
